Usage

Installation
============

This is published on PyPI.
You can install this by your package management tools.

.. tab-set::

    .. tab-item:: pip

        .. code-block:: shell

            pip install atsphinx-helper


    .. tab-item:: uv

        .. code-block:: shell

            uv add atsphinx-helper

Apply your extensions
=====================

This provides functions, classes and modules to upgrade your extensions.
It writes document for features into docstrings, so you can see :doc:`./api` for them.
